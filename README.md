### ✨ Key Features

- **📟 Automatic Calculation**: Set your target and rate; the tool instantly projects all the necessary infrastructure.
- **📊 Dynamic Graphs**: Hierarchical visualization with color-coded connections based on material types.
- **🔋 Supply Management**: Add resources you already have and watch the production chain simplify in real-time.
- **🌡️ Live Metrics**: Detailed monitoring of power consumption, heat generation, and total building count.
- **🛰️ Orbital Export**: Integrated calculation of cargo launchers required for output logistics.

### 🕹️ Quick Start Guide

1.  🎯 **Target**: Choose your final product and items per minute (IPM).
2.  📥 **Supplies**: Use the sidebar to inject base materials and reduce the required machinery.
3.  📐 **Layout**: Drag nodes to organize your flow. The camera stays fixed to allow for easy editing.
4.  ⚡ **Stats**: Check the dashboard to ensure your power grid can handle the load.

## Español

**Planifica. Optimiza. Construye.** El orquestador definitivo para tus cadenas de producción en **Starrupture**.

### ✨ Características principales

- **📟 Cálculo Automático**: Selecciona tu objetivo y el IPM; la herramienta proyecta instantáneamente toda la infraestructura necesaria.
- **📊 Grafos Dinámicos**: Visualización jerárquica con conexiones codificadas por colores según el tipo de material.
- **🔋 Gestión de Suministros**: Añade recursos que ya posees y observa cómo la cadena de producción se simplifica en tiempo real.
- **🌡️ Métricas en Vivo**: Monitorización detallada de consumo eléctrico, generación de calor y conteo total de edificios.
- **🛰️ Exportación Orbital**: Cálculo integrado de cargadores necesarios para la logística de salida.

### 🕹️ Guía de uso rápido

1.  🎯 **Objetivo**: Elige el producto final y la cantidad por minuto.
2.  📥 **Suministros**: Usa el sidebar para inyectar materiales base y reducir la maquinaria necesaria.
3.  📐 **Layout**: Arrastra los nodos para organizar tu flujo. La cámara se mantiene fija para facilitar la edición.
4.  ⚡ **Stats**: Revisa el panel de control para asegurar la viabilidad de tu red eléctrica.
